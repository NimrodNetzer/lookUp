Download & extract

Click the button above to download the zip. Extract the folder anywhere on your computer.

2

Enable Developer Mode

Open chrome://extensions in Chrome and toggle on Developer mode in the top-right corner.

3

Load unpacked

Click Load unpacked, select the extracted folder, and LookUp will appear in your extensions.

4

Get a free Groq API key

Sign up at console.groq.com (30 seconds, no card needed) and copy your API key.

5

Paste & go

Click the LookUp icon in your Chrome toolbar, paste your API key when prompted, and you're live.

6

Capture & ask

Screenshot any tab, record a lecture, or paste text — then ask LookUp to explain, quiz, summarise, or generate flashcards.

FAQ

Common questions

Everything you need to know before you install.

Your data never leaves your device except for the AI inference call to Groq's servers (the same way ChatGPT works). Screenshots and text are sent only when you explicitly ask a question — nothing is recorded passively. Groq does not store your data beyond the request.
No account needed. Just install the extension, enter a free Groq API key (takes 30 seconds to get), and you're ready. No email, no sign-up, no subscription.
Groq offers a generous free tier — more than enough for daily study use. You create a free key at console.groq.com in under a minute. LookUp uses your key directly, so you're always in control of usage and costs.
The extension UI, notes, and conversation history all work offline. AI responses require an internet connection since they call Groq's API — but your saved notes and previous conversations are always available locally.
LookUp is a Chrome MV3 extension and works on Chrome, Brave, Edge, and any Chromium-based browser. Firefox is not currently supported. A Chrome Web Store listing is coming soon — for now, install by loading unpacked from GitHub.
